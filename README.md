# ai_testing_metamorphic

メタモルフィックテスティングの練習

## お題

* 三角形の画像判定モデルを考える
    * マイヤーズの三角形問題をAIで
    * 入力された画像が、正三角形か、二等辺三角形か、不等辺三角形かを画像で判定する
    * もともとのマイヤーズの三角形問題は
        * 「3つの整数を入力に取り、その長さの辺で構成される三角形が、正三角形か、二等辺三角形か、不等辺三角形のいずれであるかを判定するプログラム、に対するテストケースにはどのようなものが考えられるか？」というもの

## 構成

* python 3.6.8
* opencv-python 4.5.3.56
